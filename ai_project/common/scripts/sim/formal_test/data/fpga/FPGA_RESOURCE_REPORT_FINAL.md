## 2. 模型架构

### 2.1 推荐模型：SAGE2-Lite-64

```
输入层 (15)
    ↓
Conv1: SAGEConv(15 → 64)  + ReLU + Dropout
    ↓
Conv2: SAGEConv(64 → 32)  + ReLU + Dropout
    ↓
MLP1: Linear(32 → 8)      + ReLU + Dropout
    ↓
MLP2: Linear(8 → 1)       + Sigmoid
    ↓
输出层 (1) — 脆弱性概率
```

### 2.2 备选模型：SAGE2-Lite-32

```
输入层 (15)
    ↓
Conv1: SAGEConv(15 → 32)  + ReLU + Dropout
    ↓
Conv2: SAGEConv(32 → 16)  + ReLU + Dropout
    ↓
MLP1: Linear(16 → 8)      + ReLU + Dropout
    ↓
MLP2: Linear(8 → 1)       + Sigmoid
    ↓
输出层 (1) — 脆弱性概率
```

### 8.2 推荐结论

**首选方案：SAGE2-Lite-64**
- 最佳精度（F1=0.9707）
- 资源占用合理（DSP48E仅44.6%）
- 适合大多数FPGA部署场景

**备选方案：SAGE2-Lite-32**
- 极致轻量化（仅34个DSP48E）
- 精度接近最优（F1=0.9614）
- 适合资源极其受限的边缘部署
