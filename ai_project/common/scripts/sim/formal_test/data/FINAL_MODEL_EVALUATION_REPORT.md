## 3. PI假阳性分析

### 3.1 三阶段级联方案

```
阶段1: 规则预过滤（消除明显FP）
  ├─ PO可达性 >= 1
  ├─ DFF数量 >= 1
  ├─ 关键路径长度 >= 3
  └─ fanout cone大小 >= 10

阶段2: SAGE3高阈值（0.75 vs 默认0.5）
  └─ 仅SAGE3分数 > 0.75的PI进入下一阶段

阶段3: PI分类器确认
  └─ 基于下游影响分析的加权评分
```

## 6. 优化策略总结

### 6.1 算法层面

| 优化项 | 效果 |
|--------|------|
| 数据采样（50%） | 训练时间减半，性能损失<1% |
| 早停机制（patience=10） | 避免无效训练，节省30-40%时间 |
| CosineAnnealingLR | 稳定收敛，避免震荡 |
| 梯度裁剪 | 防止梯度爆炸，训练更稳定 |
| 类别加权损失 | 解决类别不平衡问题 |

### 6.2 架构层面

| 优化项 | 效果 |
|--------|------|
| 减少GNN层数（3→2） | 减少计算量，提升收敛速度 |
| 减少隐藏通道（128→64→32） | 大幅减少参数和计算量 |
| INT8量化 | 内存减少75%，计算效率提升 |
| 移除Dropout（推理） | 简化硬件实现 |

### 6.3 综合效果

| 指标 | 优化前 | 优化后 | 提升 |
|------|--------|--------|------|
| 训练时间 | ~18分钟 | 2.5-6分钟 | 6-7倍 |
| F1分数 | 0.7737 | 0.9707 | +25% |
| 参数数量 | 52,577 | 6,385 | 83%减少 |
| DSP48E | 477 | 99 | 79%减少 |
